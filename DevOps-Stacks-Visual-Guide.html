Decision cheat-sheet

Pick by team size

  • Solo: GitHub Actions + Docker Compose + single VPS.
  • Small team: GitHub Actions + Terraform + Railway/Fly.
  • Medium team: ArgoCD + Kubernetes + Terraform + Grafana.
  • Large org: Multi-cloud + service mesh + platform team + Backstage.

Pick by constraint

  • Cheapest: Hetzner + Coolify + GitHub Actions free.
  • Fastest deploy: Kamal + GitHub Actions.
  • Most reliable: Managed K8s + Datadog + multi-region.
  • Compliance: AWS/Azure + Vault + OPA + audit logs.

Anti-patterns to avoid

  • Kubernetes before 10 engineers.
  • Microservices before product-market fit.
  • Manual deploys to production.
  • No monitoring until incidents happen.
  • Storing secrets in git.
  • Skipping staging environments.
