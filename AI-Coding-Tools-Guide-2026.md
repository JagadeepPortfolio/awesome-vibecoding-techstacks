## 8. MCP (Model Context Protocol) Ecosystem

MCP was introduced by Anthropic in late 2024. By 2026 it is the de facto standard for connecting LLMs to external tools, adopted by OpenAI, Microsoft, Google, and Amazon. 12,000+ MCP servers exist across GitHub, npm, PyPI, and registries like Smithery, Glama, and PulseMCP.

| Server | Stars | What It Does | Source |
|--------|-------|-------------|--------|
| **GitHub MCP Server** | 28,300+ | Official first-party server (GitHub + Anthropic). 51 tools for repo management, issues, PRs, CI/CD. | https://github.com/modelcontextprotocol/servers |
| **Filesystem** | - | Read/write local files. MIT licensed. | Official |
| **PostgreSQL** | - | Query databases directly from AI agents. MIT licensed. | Official |
| **Brave Search** | - | Live web search for AI agents. | Official |
| **Playwright** | - | Browser automation. MIT licensed. | Official |
| **Slack** | - | Channel management, messaging, search conversations. | https://github.com/korotovsky/slack-mcp-server |
| **Google Drive** | - | File access and management. | Official |
| **Notion** | - | Page and database operations. | Community |
| **Sentry** | - | Error monitoring integration. | Official |
| **Memory** | - | Persistent memory/knowledge for AI agents. | Official |
| **Puppeteer** | - | Browser automation (alternative to Playwright). | Community |
| **SQLite** | - | Local database operations. | Official |

### MCP Infrastructure

- **TypeScript SDK** v1.27.1 (Feb 2026) - 34,700+ dependent projects
- **Python SDK** - mature and widely adopted
- **Registries**: Smithery, Glama, PulseMCP
- **2026 Roadmap**: Enterprise readiness -- audit trails, corporate auth, security hardening
- **Security concern**: ~1,000 MCP servers exposed on public internet with no auth controls (Bitsight report)
